# 堆排序

堆排序（Heap Sort）是一种基于堆这种数据结构的排序算法，具有 O(n log n) 的时间复杂度，且可以原地排序（不需要额外空间），是非常经典的排序方法。

## ✨ 堆排序的核心思路

以升序排序为例（使用最大堆）：

1. 构建最大堆：将无序数组变成一个最大堆，保证堆顶是最大值。
2. 交换堆顶和末尾元素：最大元素移到数组末尾。
3. 堆的大小减 1，重新堆化：让剩余部分继续是最大堆。
4. 重复步骤 2-3，直到只剩一个元素。

## 🧠 特点总结

| 特性    | 值             |
| ----- | ------------- |
| 时间复杂度 | O(n log n)    |
| 空间复杂度 | O(1)（原地排序）    |
| 稳定性   | ❌ 不稳定排序       |
| 排序方式  | 选择排序（不依赖比较顺序） |
